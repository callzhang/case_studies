挑战一：IP 孵化及运营涉及链条长，涉及协同团队人员多，如何高效实现 IP 挖掘运营？​

IP 孵化全流程落地飞书，打造收入破十亿爆款艺术家 IP​

IP 运营与创意设计是推动泡泡玛特发展的关键因素，泡泡玛特在全球范围内挖掘极具潜力的潮流艺术家和设计师，通过成熟的 IP 运营体系，打造出备受欢迎的潮流文化 IP 形象。2023 年，头部 IP SKULLPANDA、MOLLY 收入均突破 10 亿人民币。从国内到海外，泡泡玛特的业务范围越来越大，也更需要高效的数字化工具来满足复杂协同的需求。​

拿艺术家 IP 来说，整个跟艺术家合作的链条并不短，会涉及到很多的部门。从前期的孵化、中期的产品开发到后期的市场营销，信息同频非常重要。通过飞书会议和共享文档，让不见面这件事变得没有那么难：​

  * 产品开发进度管理：用飞书的多维表格管理 IP 孵化运营全流程，确保全链路所有角色可清晰了解各产品的开发进展。​



  * 飞书云文档保密能力：在近一年的产品开发周期过程中，IP 信息保密是非常重要的事情，若 IP 上市前被提前泄露，将直接影响 IP 上市的成功。因此，泡泡玛特将 IP 孵化及运营全流程信息都用飞书云文档承载，并做好权限管控与下载记录留痕。在与外部艺术家沟通时，泡泡玛特要求他们也用飞书，最大程度确保信息安全。 ​



  * 海外艺术家跨语言沟通“丝滑”：泡泡玛特在海外也挖掘了来自美国、泰国等多个国家极具潜力的潮流艺术家和设计师，与艺术家和设计师沟通协作时也常在飞书进行，在飞书群或飞书会议交流时，各自说母语，飞书可以实时丝滑完成翻译。​



  * IP 产品知识库：泡泡玛特产品部门通过飞书知识库将产品信息沉淀下来，形成产品说明小册子，赋能一线门店/客服团队，确保一线团队掌握产品 IP 信息。对于新人说来，准确地将 IP 名称念对，是一件不容易的事情，目前泡泡对多个 IP 名称录音并上传到飞书云文档，形成知识库，更好地为一线新人学习。​

门店零售：一张多维表格支撑 300 多家门店巡检​

门店是泡泡玛特跟顾客最直接的连接方式，占到了整个中国业务的一半以上。为确保为顾客提供优质购物体验，泡泡玛特每天对全国 300 多家门店进行超过 20 余个检查项的详细巡检，覆盖卫生管理、服务水平、货品管理、陈列管理等多个板块。巡检门店多、标准细，如何提升巡检效率和质量成为提升门店管理水平的关键。​

现在用飞书巡店，已经成了泡泡玛特的常态。从前期标准的统一，到整个巡店维度的统一，都是通过飞书来规划和确保。通过飞书实现的巡检流程线上化，无需投入开发资源，就能根据自己的需求，灵活定制专属巡检小程序，员工在移动端也能灵活操作。现在，每一位中高层的管理者都是在飞书上开展巡检工作。​

  * 手机随时开启整改：负责巡检的店员或店长每天打开飞书，就可以在定制的工作台中看到巡检小程序。进入小程序后，巡检任务入口清晰地展示了有多少巡检任务和整改问题亟待处理，巡检人只需要逐个点开，就能像填写问卷一样轻松填好每项内容并随时上传。​



  * 整改任务实时下发：如果在巡检中遇到需要整改的问题，小程序里配备了“随手拍问题”。 只需要简单输入对问题的描述，附上巡检现场照片，再@相关的问题负责人，就可以创建整改问题。​



  * 整改结果确保闭环：整改问题还支持“分享到聊天”和“催促整改”，问题的整改人不仅会及时收到整改提醒，相关管理人员也可以及时触达到问题发生时间节点和细节，当一个巡检任务需要多个部门协调推进的时候，参与协同者也可以随时跟进。 ​

泡泡玛特使用多维表格搭建了直播排班、商品销售数据分析看板，助力每天 120+ 小时同时直播有条不紊：​
