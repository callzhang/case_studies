用飞书做游戏研发，提升组织协同能力，开发世界级 IP​

作为领先的全球化 IP 游戏生态公司，中手游始终秉承“正直守信、透明坦诚、深度思考、长期主义”的价值观，和“用热爱铸造精品”的使命，为全球玩家提供精品 IP 游戏，打造极具竞争力的 IP 游戏生态体系。截止目前，中手游已成功推出了《新仙剑奇侠传》、《航海王强者之路》、《火影忍者-忍者大师》、《倚天屠龙记》、《择天记》、《传奇世界之雷霆霸业》、《龙珠觉醒》、《家庭教师》、《轩辕剑：剑之源》、《新射雕群侠传之铁血丹心》、《航海王热血航线》、《斗罗大陆-斗神再临》、《真·三国无双 霸》和《仙剑奇侠传七》、《大富翁10》等数十款精品 IP 游戏。​

为了持续向粉丝创造精品内容和互动体验，组织管理必须跟上脚步。为此，中手游深度使用飞书，建立起更加标准、敏捷、高效的游戏研发流程，提升团队协同效率，增强组织力，从而开发出优质的世界级 IP ，领跑移动游戏行业第一棒。​

  * 用云文档评审产品，实现会前讨论前置，返工率降低 50%​



对于游戏公司来说，打造一款具备创新、有趣、优质等特质的现象级游戏，在前期设计过程中，评审是重要一环，只有产品创意评审通过之后，才会进入开发阶段。​

使用飞书前，在策划部门完成产品文档撰写之后，需要邀请前后端开发、美术、质控等相关参与人员进行评审，然后从各自角度提出存在的问题，再进一步调整。由于没有共享文档，大家只能在评审会议上当面讨论，如果遇到某一评审人员忙别的事情，容易导致评审质量不高；同时，由于无法前置讨论，一次评审过后还会出现很多问题，又得进行二次评审，对开发进度不利。​

使用飞书后，基于飞书云文档强大的协同共创能力，只需把文档发到开发小群，所有人可以同时查看并进行编辑修改，有问题可以直接评论，还能 @ 相关策划人员，实现异步沟通。现在，很多设计问题在会前就能高效处理掉，进行更好地思考，会议质量大幅提升，减少了二次评审的情况，返工率降低 50%。​
